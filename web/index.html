LimeChat

[Japanese]

What's LimeChat ?

LimeChat is an IRC client for Mac OS X.

Features

  • One window for multiple servers.
  • Rich keyboard shortcuts for your comfortable operations.
  • Fast and stable.

Requirements

  • Mac OS X Tiger and over

Install

Lion / Snow Leopard

  1. Download LimeChat_2.29.tbz.
  2. Extract the downloaded package, and copy LimeChat.app to your Applications folder.

Leopard

  1. Download LimeChat_2.26.tbz.
  2. Extract the downloaded package, and copy LimeChat.app to your Applications folder.

Tiger

  1. Download RubyCocoa-0.13.2.1-OSX10.4universal.dmg, and install it.
  2. Download LimeChat 0.20 executable dmg.
  3. Open the downloaded dmg, and copy LimeChat.app to your Applications folder.

Shortcut keys

Tab Complete nickname.
Ctrl+Tab
Opt+Space		 Move to the next unread channel.
Ctrl+Shift+Tab
Opt+Shift+Space		 Move to the previous unread channel.
Opt+Tab Move back to the previously selected channel.
Cmd+Up/Down
Cmd+Opt+Up/Down		 Move to the previous/next active channel.
Ctrl+Up/Down Move to the previous/next channel.
Cmd+Opt+Left/Right Move to the previous/next active server.
Ctrl+Left/Right Move to the previous/next server.
Cmd+0..9 Move to the 1st .. 10th channel in the current server.
Cmd+Ctrl+1..0 Move to the 1st .. 10th server.
Ctrl+Enter Send a notice message.

Change log

This change log is also available in RSS.

2.29 (2012.01.15)
  • Added a setting for double click actions in user list.
  • Support ZNC capability to send server times when SASL is enabled.
  • Changed ustream host name to chat1.ustream.tv.
  • Show SSL errors more clearly.
  • Recognize afp:// URLs.
2.28 (2011.08.19)
  • Fixed a redraw issue in log view.
  • Do not set auto join when adding a channel with /join.
2.27 (2011.08.09)
  • Make marked scroller work in Lion.
  • Fixed an issue that Growl notifications didn't work on some environments.
  • Allow single quotation in URLs.
2.26 (2011.07.26)
  • Full screen support for Lion.
  • Catch up on updates of Gist.
  • Added main window menu item to main menu.
  • Changed button caption in paste dialog from "Send in Channel" to "Send".
  • Fixed a bug where 3 fingers double tap in the other app caused wrong hot key event.
2.25 (2011.07.22)
  • Improved image URL parser.
  • Don't attempt to rename nicks if the new nick is the same.
  • Fixed an issue that adding highlight word didn't work well on Lion.
  • Don't use custom scrollers on Lion.
2.24 (2011.06.05)
  • Fixed a bug in parsing mode.
  • Updated Limelight theme.
  • Use chat01.ustream.tv instead of chat1.ustream.tv.
  • Changed caption for Nickserv password.
2.23 (2011.04.19)
  • Fixed the auto scroll problem.
  • Added pong interval option.
  • Refined preferences dialog.
